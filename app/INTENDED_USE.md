# Intended Use Statement

The Neuro-Symbolic Anemia Assistant is a clinical decision support (CDS)
prototype intended for educational and research purposes only.

## Intended Users
- Medical students
- Clinical researchers
- Healthcare technology developers

## Intended Use
The system provides:
- Risk estimation for anemia using machine learning
- Rule-based clinical pattern recognition
- Transparent explanations for educational review

## Limitations
- Not a diagnostic device
- Not validated for clinical use
- Does not replace clinician judgment
- Uses synthetic training data

## Regulatory Status
This system is a non-medical, investigational CDS prototype and is not
FDA-cleared, CE-marked, or approved for clinical decision-making.

Clinical decisions must not be made based solely on this system.
